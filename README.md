# 🌱 Growing Awareness

**Your Recovery Companion** — a free, private, mobile-friendly web app for people on a gambling-free journey.

🔗 **Live app:** [ga.shayde.me](https://ga.shayde.me)

---

## What It Does

Growing Awareness helps you track and celebrate your gambling-free streak, stay motivated during tough moments, and keep the right people one tap away when you need them most.

All data is stored **only on your device** — nothing is sent to a server or shared anywhere.

---

## Features

### 🏠 Home — Streak Tracker
- Circular progress ring showing your current gamble-free day count
- **17 progression levels** from 🌱 Seedling (Day 0) to ☀️ Eternal Sun (30 years)
- Level progress bar with a motivational message at each level
- Next milestone countdown (e.g. "5 days until 🌿 Sprout")
- Set or edit your clean date at any time
- Reset button with a compassionate, judgment-free confirmation
- Share your progress

### 🌸 Daily Check-In
- Pick an emoji mood each day (once per day)
- **Garden view** — every check-in day grows a flower; tap any flower to see its date

### 📞 Contacts
- Save names and phone numbers of people you trust
- One-tap calling directly from the app
- Add, edit, and delete contacts at any time

### 🏆 Badges
- 13 milestone badges earned automatically as your streak grows
- From **Day One** 🌅 all the way to **Twenty-Five Years** 🔱

### 💪 Stay Strong
- Rotating motivational quotes from well-known figures
- **National Problem Gambling Helpline** prominently displayed: **1-800-522-4700** (free, confidential, 24/7)
- **Urge Toolkit** — four quick, evidence-based strategies to ride out a craving:
  - Delay 15 minutes
  - Move your body
  - Call someone
  - Write it down

## Tech Stack

| Layer | Details |
|---|---|
| Language | HTML5, CSS3, Vanilla JavaScript |
| Storage | `localStorage` (client-side only) |
| Dependencies | None — zero external libraries or frameworks |
| Hosting | GitHub Pages |

The entire app is a single `index.html` file — no build step required.

---

## Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/ShaydeNofziger/GrowingAwareness.git
   ```
2. Open `index.html` in any modern browser.

That's it — no install, no server needed.

---

## Privacy

- **No accounts.** No sign-up, no login.
- **No network requests.** The app never sends your data anywhere.
- **Device-only storage.** Everything lives in your browser's `localStorage`.
- Clearing your browser data will erase your streak and contacts, so take note of your clean date before doing so.
